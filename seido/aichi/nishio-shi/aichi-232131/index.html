西尾市障害者扶助料

身体・知的・精神の各手帳所持者に等級別月額1,000〜4,000円を年2回（9月・3月）に支給する西尾市単独の現金給付。令和8年4月から軽度等級（身4-6級／C／精神3級）が月2,000円→1,000円に減額、激変緩和の「障がい者生活支援給付金」（年額36,000円/9,000円）が令和8年10月頃支給予定。国の特別障害者手当・障害児福祉手当・経過的福祉手当・愛知県在宅重度障害者手当の受給資格者は対象外（併給不可）。

金額・負担額

【令和8年4月以降】身1-2級／知的A／精神1級: 月額 4,000円 / 身3級／知的B／精神2級: 月額 3,000円 / 身4-6級／知的C／精神3級: 月額 1,000円（令和8年3月までは2,000円→4月から1,000円に減額）（年2回 9月・3月、申請月の翌月分から）

対象
西尾市内に住所を有する身体・知的・精神の各手帳所持者。65歳未満で障害者となった方が対象（平成30年度から65歳以上で新規に障害者となった方は対象外）。国の特別障害者手当・障害児福祉手当・経過的福祉手当・愛知県在宅重度障害者手当の受給資格がある方は対象外（併給不可、公式明文）。

西尾市は令和8年4月から軽度等級（身4-6級／知的C／精神3級）の月額を2,000円→1,000円に減額する改定を実施。激変緩和として令和8年度のみ「障がい者生活支援給付金」（年額36,000円または9,000円）を令和8年10月頃支給予定で、減額分の一部または全部が補填されます。令和9年度以降の継続有無は公式未記載・要照会のため、長期的な家計設計には注意が必要です。

この制度をくわしく

この制度とは

西尾市が市単独で実施している障害者向け現金給付制度です。公式名称は「西尾市障害者扶助料」、根拠条例名は推定「西尾市障害者扶助料支給条例」（条例本文は要照会）。

愛知県内の多くの市町村は、独自に「障害者扶助料」「心身障害者手当」等の現金給付制度を設けています。これらは国の特別障害者手当（令和8年4月以降月額30,450円）や障害児福祉手当（令和8年4月以降月額16,560円）、さらに愛知県在宅重度障害者手当（県単独、月6,750円）とも別枠で、市町村が独自財源で実施する現金給付です。

西尾市の本扶助料の特徴は、令和8年4月から軽度等級が減額される改正（4-6級／C／精神3級が月2,000円→1,000円）、激変緩和措置として「障がい者生活支援給付金」を令和8年度に支給（年額36,000円/9,000円・令和8年10月頃）、平成30年度から65歳以上で新規に障害者となった方は対象外、国手当（特別障害者手当・障害児福祉手当・経過的福祉手当）と県在宅重度障害者手当との併給不可（受給資格者は本扶助料の対象外）、所得制限は本人のみで判定という設計です。

激変緩和措置（令和8年度のみ）

令和8年4月の減額対象者（身4-6級／知的C／精神3級）への激変緩和として、市が「障がい者生活支援給付金」を令和8年度に支給:

- 該当者に年額36,000円または9,000円

- 令和8年10月頃支給予定

これにより、令和8年度内は減額分の一部または全部が補填されます（令和9年度以降の継続有無は公式未記載・要照会）。

対象となる方

西尾市内に住所を有する以下のいずれかの方:

- 身体障害者手帳1〜6級

- 療育手帳A・B・C

- 精神障害者保健福祉手帳1〜3級

65歳未満で障害者となった方が対象で、平成30年度から65歳以上で新規に障害者となった方は対象外となります。

支給対象外（公式明文）

- 平成30年度以降、65歳以上で新規に障害者となった方

- 国の特別障害者手当の受給資格者

- 国の障害児福祉手当の受給資格者

- 国の経過的福祉手当の受給資格者

- 愛知県在宅重度障害者手当の受給資格者

国手当・県手当の受給資格者は本扶助料の対象外となるため、本扶助料を選ぶか国・県手当を選ぶかの二者択一となります。

在宅／施設の扱い

公式ページに在宅要件・施設入所除外の明文記載なし（公式未記載・要照会）。施設入所中の取扱いは福祉課（0563-65-2113）にご確認ください。

所得制限・支給停止

所得制限は本人のみ（公式明文）。具体的限度額は公式ページに非掲載で条例または規則別表に規定されていると推定されますが、申請前に福祉課（0563-65-2113）に必ず確認してください。

他の手当との併用

- 国の特別障害者手当（令和8年4月〜月額30,450円）: 併給不可（公式明文、受給資格者は本扶助料対象外）

- 国の障害児福祉手当（令和8年4月〜月額16,560円）: 併給不可（公式明文、同上）

- 国の経過的福祉手当: 併給不可（公式明文、同上）

- 愛知県在宅重度障害者手当（月6,750円）: 併給不可（公式明文、受給資格者は本扶助料対象外）

- 国の特別児童扶養手当（1級56,800円・2級37,830円）: 公式に明示記載なし、保護者支給のため別枠と読めるが要照会

- 愛知県重度障害者医療費助成（医療費）: 別制度のため併用可能

- 児童手当: 別制度のため併用可能

金額比較: 国の特別障害者手当（月30,450円）・障害児福祉手当（月16,560円）の方が本扶助料（最大月4,000円）より大幅に高額のため、併給不可の場合は国手当を選ぶのが経済的合理的です。本扶助料の主な対象は、国手当の認定基準を満たさない軽度〜中度の手帳所持者となります。

申請方法

申請窓口は西尾市 健康福祉部 福祉課 障がい福祉担当（TEL 0563-65-2113）または各支所地域・住民担当。必要書類は障害者扶助料認定申請書（Word/PDF/記入例PDFを公式DL可）、障害者手帳、本人名義の振込先口座、マイナンバーカードまたは通知カード（公式明文）。代理申請時は委任状必須（公式明文）。電子申請の可否は公式未記載で、窓口提出が前提と読めます（要照会）。

このお子さんが対象となるか

- 身1-2級・知的A・精神1級のお子さんで国手当未受給: 月額4,000円（年額48,000円）が対象となる可能性があります。

- 身3級・知的B・精神2級のお子さん: 月額3,000円（年額36,000円）が対象となる可能性があります。

- 身4-6級・知的C・精神3級のお子さん: 令和8年3月までは月2,000円、令和8年4月から月1,000円（年額12,000円）に減額予定。令和8年度は障がい者生活支援給付金（年額36,000円または9,000円）が令和8年10月頃支給予定で、減額分の一部または全部が補填されます。

- 国の障害児福祉手当受給中のお子さん: 対象外です（併給不可、公式明文）。金額の大きい国の障害児福祉手当（月16,560円）を選ぶのが経済的合理的。

申請の手順

  1. 1西尾市 健康福祉部 福祉課 障がい福祉担当（TEL 0563-65-2113）または各支所の窓口で「障害者扶助料認定申請書」を入手（Word/PDF/記入例PDFを公式DL可）
  2. 2障害者手帳、本人名義の振込先口座、マイナンバーカードまたは通知カードを持参し窓口で申請（代理申請時は委任状必須、電子申請可否は公式未記載のため事前電話確認推奨）
  3. 3市が等級・在住・所得（本人のみ判定、具体額は公式未記載）・65歳未満要件・国県手当非受給を審査
  4. 4認定後、申請月の翌月分から支給対象。年2回（9月・3月）に半年分まとめて指定口座へ後払い振込

知っておくと役立つこと

  • ●令和8年4月から軽度等級（身4-6級／知的C／精神3級）が月2,000円→1,000円に減額（半額）。減額対象者は令和8年4月以降の家計影響を事前確認推奨
  • ●激変緩和措置「障がい者生活支援給付金」を令和8年度に支給（年額36,000円または9,000円・令和8年10月頃）。減額対象者は別途給付金で補填されるが、令和9年度以降の継続有無は公式未記載・要照会
  • ●国手当・県手当との併給不可（公式明文）。本扶助料の最大月4,000円と国の特別障害者手当（月30,450円）の差が大きいため、併給不可の場合は国手当を選ぶのが経済的合理的
  • ●平成30年度から65歳以上で新規に障害者となった方は対象外。65歳到達前に手帳取得した方は継続して対象（経過措置の有無は要照会）
  • ●所得制限の判定対象は本人のみ（公式明文、配偶者・世帯所得は判定軸ではない）
  • ●代理申請時は委任状必須（公式明文）。家族が代わりに申請する場合も委任状を用意

この制度と一緒に検討されることが多い制度

  • 手当国の制度

    特別障害者手当

    精神または身体に著しく重度の障害があり、日常生活において常時特別の介護を必要とする20歳以上の在宅の方に支給されます。障害児福祉手当の20歳以降版。

  • 手当国の制度

    障害児福祉手当

    重度の障害があり日常生活で常時介護が必要な在宅の20歳未満の児童に支給されます。特別児童扶養手当との併給が可能です。

  • 手当国の制度

    特別児童扶養手当

    身体または精神に中度以上の障害がある20歳未満の児童を養育する方に支給されます。
