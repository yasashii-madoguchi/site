知多市障害者福祉手当

身体・療育・精神の各手帳所持者に課税状況×手帳種別×等級により月額1,600〜6,000円を年2回（9月・3月）に支給する知多市単独の現金給付。手帳3種×等級×課税区分の細分化マトリクスで身5-6級・療育C・精神3級まで全等級対象。

金額・負担額

【身体・課税/非課税】1級 4,000/6,000円・2級 3,000/4,100円・3級 2,500/3,100円・4級 1,800/2,000円・5-6級 1,600/1,800円 【療育・課税/非課税】A 4,000/6,000円・B 2,500/3,100円・C 1,600/1,800円 【精神・課税/非課税】1級 4,000/6,000円・2級 2,500/3,100円・3級 1,600/1,800円（年2回 9月・3月支給）

対象
知多市内に住所を有する身体・療育・精神の各手帳所持者。身5-6級・療育C・精神3級まで全等級対象（市町村単独手当としては珍しく広範）。

注意事項
課税/非課税で支給額が変わる二段階方式（所得制限による不支給ではなく課税状況による減額）。手当の間で、また要介護高齢者福祉手当との間で併給制限あり（公式明文）。根拠条例は公式に明示記載なく要綱運用と推定（要照会）。所得制限の判定指標（本人課税のみか同一世帯か、扶養親族数による加算等）は公式未記載・要照会。国の特別障害者手当・障害児福祉手当・特別児童扶養手当との具体的な併給可否、愛知県在宅重度障害者手当との併給、年齢要件、施設入所中の取扱い、65歳以上新規取得者の扱い、電子申請可否は公式に明示記載なく要照会（TEL 0562-36-2650）。

課税状況による減額方式の特徴

他市の障害者手当（所得制限ありのタイプ）

  • ・所得制限超過で支給停止（不支給）
  • ・判定基準: 本人または世帯の所得限度額
  • ・限度額が公開される市と未公開の市あり
  • ・所得が高ければ完全に対象外

知多市障害者福祉手当（このページの制度）

  • ・課税状況による減額方式（不支給ではない）
  • ・課税世帯も金額は減るが引き続き対象
  • ・身5-6級・療育C・精神3級まで全等級対象
  • ・判定指標（本人/世帯）は公式未記載・要照会

知多市の本手当は所得制限超過による不支給ではなく、課税世帯にも減額しつつ支給を続ける設計が特徴。所得が高い世帯でも非課税の身1級は月4,000円（年48,000円）の対象となる可能性があり、所得制限ありの市と比べて受給機会が広い設計です。判定指標（本人課税のみか世帯か）は公式未記載のため、世帯所得が高い場合の本人非課税適用は要照会。

この制度をくわしく

この制度とは

知多市が市単独で実施している障害者向け現金給付制度です。公式名称は「障害者福祉手当」、根拠条例は公式に明示記載なく要綱運用と推定されます（要照会）。

愛知県内の多くの市町村は、独自に「障害者扶助料」「心身障害者手当」等の現金給付制度を設けています。これらは国の特別障害者手当（令和8年4月以降月額30,450円）や障害児福祉手当（令和8年4月以降月額16,560円）、さらに愛知県在宅重度障害者手当（県単独、月6,750円）とも別枠で、市町村が独自財源で実施する現金給付です。

知多市の本手当の特徴は、手帳3種×等級×課税区分の細分化マトリクス（合計約24パターン）、月額1,600〜6,000円の幅広い金額レンジ、身5-6級・療育C・精神3級まで全等級対象で市町村単独手当としては珍しく広範な対象設計、課税状況による減額方式（所得制限超過で支給停止する設計ではなく、課税世帯にも減額しつつ支給する設計）です。

対象となる方

知多市内に住所を有する以下のいずれかの方:

- 身体障害者手帳1〜6級

- 療育手帳A・B・C

- 精神障害者保健福祉手帳1〜3級

身5-6級・療育C・精神3級まで全等級対象で、市町村単独手当としては珍しく広範な対象設計です。

在宅／施設の扱い

公式ページに在宅要件・施設入所除外の明文記載なし（公式未記載・要照会）。同ページの特別障害者手当・障害児福祉手当・経過的福祉手当・在宅重度障害者手当には施設入所・長期入院除外規定がある一方、市の本手当には記載がないため、扱いは要照会です。

所得制限・支給停止

課税/非課税で支給額が変わる二段階方式（公式明文）。所得制限による不支給ではなく、課税状況による減額を採用しているため、課税世帯でも金額は減るものの引き続き対象となります。判定指標（本人課税のみか同一世帯か、扶養親族数による加算等）は公式未記載・要照会です。

他の手当との併用

- 市内の他の手当および要介護高齢者福祉手当との間で併給制限あり（公式明文）

- 国の特別児童扶養手当（1級56,800円・2級37,830円、令和8年4月〜）: 公式に併給制限の明示なし、要照会

- 国の障害児福祉手当（令和8年4月〜月額16,560円）: 公式に併給制限の明示なし。同ページに国・市の手当が並列記載されており、市単独手当が併給制限対象に入るかは記載が曖昧、要照会

- 国の特別障害者手当（令和8年4月〜月額30,450円）: 公式に併給制限の明示なし、要照会

- 愛知県在宅重度障害者手当（月6,750円）: 同ページ並列記載だが具体的な併給関係は未記載、要照会

- 愛知県重度障害者医療費助成（医療費）: 別制度のため併用可能

- 児童手当: 別制度のため併用可能

申請方法

申請窓口は知多市 福祉子ども部 福祉課（〒478-8601 知多市緑町1番地、TEL 0562-36-2650 手当担当直通）、受付時間は月〜金9:00〜16:00（閉庁日除く）。必要書類は公式ページ・FAQ・福祉課組織ページとも明記なしで、一般的には障害者手帳・本人名義口座・印鑑・マイナンバー等が想定されますが要照会です。電子申請・郵送可否は公式未記載で、窓口申請のみが想定されます（要照会）。支給開始日（申請月の翌月分からか）は公式未記載・要照会です。

このお子さんが対象となるか

- 非課税世帯で身1級・療育A・精神1級のお子さん: 月額6,000円（年額72,000円）が対象となる可能性があります。

- 非課税世帯で身2級・療育B・精神2級のお子さん: 月額3,100〜4,100円（年額37,200〜49,200円）が対象となる可能性があります。

- 非課税世帯で身5-6級・療育C・精神3級のお子さん: 月額1,800円（年額21,600円）が対象となる可能性があります（他市が対象外とする等級でも対象）。

- 課税世帯の場合: 上記の約3分の2の金額（月1,600〜4,000円）となります。

- 要介護高齢者福祉手当受給中の方: 公式明文で併給制限ありのため要照会です。

  3. 3市が等級・在住・課税状況・他の手当併給状況等を審査
  4. 4認定後、年2回（9月・3月）に指定口座へ振込（支給開始日・後払い方式・年内何か月分か等は公式未記載のため要照会）

知っておくと役立つこと

  • ●身5-6級・療育C・精神3級まで全等級対象は市町村単独手当としては珍しい広範な設計（多くの市は身3-4級または療育B程度で打ち切り）
  • ●所得制限超過で不支給ではなく、課税世帯にも減額しつつ支給（課税世帯と非課税世帯で金額が変わる二段階方式）。所得が高くても支給対象になる利点
  • ●課税/非課税の判定指標（本人のみか世帯か）が公式未記載。世帯の他の家族の所得が高い場合の扱いは要照会
  • ●要介護高齢者福祉手当との併給制限あり（公式明文）。65歳以上の方は他市と異なる注意点
  • ●申請書類の詳細・電子申請可否は公式未記載。同ページの情報量は他自治体と比較してかなり薄いため、申請前に必ず福祉課（0562-36-2650）に電話確認推奨
  • ●支給は年2回（9月・3月）。何か月分の後払い方式かは公式未記載のため要照会

この制度と一緒に検討されることが多い制度

  • 手当

    特別障害者手当

    精神または身体に著しく重度の障害があり、日常生活において常時特別の介護を必要とする20歳以上の在宅の方に支給されます。障害児福祉手当の20歳以降版。

  • 手当

    障害児福祉手当

    重度の障害があり日常生活で常時介護が必要な在宅の20歳未満の児童に支給されます。特別児童扶養手当との併給が可能です。

  • 手当

    特別児童扶養手当

    身体または精神に中度以上の障害がある20歳未満の児童を養育する方に支給されます。
