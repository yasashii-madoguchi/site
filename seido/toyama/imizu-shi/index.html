射水市で使える障害児向けの制度・手当

富山県射水市にお住まいの方が利用できる、障害のある子ども向けの制度・手当の一覧です。

こども医療費助成

通院: 18歳年度末まで / 入院: 18歳年度末まで / 所得制限なし

詳細を見る →

射水市の独自制度

手当（市区町村独自）

射水市心身障がい者（児）福祉金

身体1・2級/療育A/精神1級: 月額 2,000円 / 身体3級/療育B/精神2級: 月額 1,500円 / 身体4級/精神3級: 月額 1,100円

射水市独自の手当。全等級対象。

富山県の独自制度

医療費助成（都道府県独自）

重度心身障害者等医療費助成

重度（身体1-2級・療育A・精神1級）: 自己負担なし / 65歳以上の中度: 一部自己負担あり

身体手帳1-2級、療育手帳A、精神手帳1級の方の医療費自己負担がなくなります（65歳未満）。

全国共通の制度

手当

特別児童扶養手当

1級（重度）: 月額58,450円 / 2級（中度）: 月額38,930円

身体または精神に中度以上の障害がある20歳未満の児童を養育する方に支給されます。

手当

障害児福祉手当

月額 16,560円

重度の障害があり日常生活で常時介護が必要な在宅の20歳未満の児童に支給されます。特別児童扶養手当との併給が可能です。

手当

児童手当

3歳未満: 月額15,000円 / 3歳〜高校生年代: 月額10,000円 / 第3子以降: 全年齢 月額30,000円

障害の有無にかかわらず全児童が対象。令和6年10月から所得制限が撤廃されました。

療育・福祉サービス

児童発達支援

非課税世帯: 0円 / 一般1: 上限月額4,600円 / 一般2: 上限月額37,200円 / 3〜5歳: 無料

未就学児（0〜6歳）向け。日常生活の基本動作や集団適応の訓練を受けられます。3〜5歳は無償化対象。

療育・福祉サービス

放課後等デイサービス

非課税世帯: 0円 / 一般1: 上限月額4,600円 / 一般2: 上限月額37,200円

就学児（6〜18歳）向け。放課後や休日に療育・社会性向上の支援を受けられます。

医療費助成

自立支援医療（育成医療）

自己負担1割（所得に応じた上限月額: 0円〜20,000円）

身体障害のある児童の手術等の治療費が、3割→1割に軽減されます。

医療費助成

小児慢性特定疾病医療費助成

自己負担2割（上限月額: 0円〜15,000円、所得区分による）

国が指定する801の慢性疾病（16疾患群）が対象。医療費の自己負担が3割→2割に軽減され、上限額が設定されます。

補装具・用具

補装具費支給制度

自己負担1割（上限月額37,200円）

義肢、装具、車椅子、補聴器、眼鏡等の購入・修理費用が支給されます。令和6年4月から障害のある子どもの所得制限が撤廃されました。

補装具・用具

日常生活用具給付

自己負担1割（所得制限撤廃済み）

たん吸引器、ネブライザー、ストーマ用装具、紙おむつ、住宅改修費等が支給されます。

教育

特別支援教育就学奨励費

学用品費・通学費・給食費・修学旅行費等を補助（所得区分により補助割合が異なる）

特別支援学校・特別支援学級・通級指導教室に通う児童の保護者に、学用品費・通学費・給食費等を補助します。

税制優遇

所得税の障害者控除

障害者控除: 27万円 / 特別障害者控除: 40万円 / 同居特別障害者: 75万円

確定申告または年末調整で障害者控除を申告すると、所得税・住民税が軽減されます。

福祉サービス

短期入所（ショートステイ）

自己負担1割（食費・光熱水費は別途実費）

介護者の病気・冠婚葬祭・介護疲れ等の際に、お子さんを施設に短期間預けることができます。

福祉サービス

移動支援（ガイドヘルプ）

自己負担1割（自治体により異なる）

外出時の移動を支援するヘルパーが付き添います。余暇活動等に利用できます。

保険

ぜんち共済（障害者専門の保険）

年額 22,000円（年齢・性別問わず定額）

知的障害・発達障害・ダウン症・てんかん・身体障害・精神障害のある方が加入できる専門保険です。一般の保険に入りにくい方向けです。

割引・減免

NHK放送受信料の減免

全額免除（非課税世帯+手帳所持） / 半額免除（視覚・聴覚障害、身体1-2級、療育A、精神1級の世帯主）

障害者手帳所持者のいる世帯は、NHK受信料が全額免除または半額免除になります。

割引・減免

有料道路通行料金の割引

通行料金が半額

障害者本人またはその介護者が運転する場合、有料道路の通行料金が半額になります。

割引・減免

公共交通機関の運賃割引

運賃が半額（1種: 本人+介護者 / 2種: 本人のみ・片道101km以上）

JR・私鉄・バス等で障害者割引が適用されます。手帳1種は本人+介護者が半額、2種は本人のみ半額（片道101km以上）。

割引・減免

携帯電話料金の障害者割引

基本料金の割引等（各キャリアにより異なる）

NTTドコモ・au・ソフトバンク等で障害者向けの割引プランがあります。

割引・減免

自動車税・軽自動車税の減免

全額免除（上限あり、都道府県により異なる）

障害者本人または生計同一者が所有する自動車の自動車税・軽自動車税が減免されます。

年金

障害基礎年金（20歳前傷病）

1級: 月額 88,260円（年額 1,059,125円） / 2級: 月額 70,608円（年額 847,300円）

20歳前に初診日がある病気やケガで障害の状態にある方に、20歳から支給される年金です。保険料の納付要件は問われません。

保険

心身障害者扶養共済制度

加入者（保護者）が死亡等した場合、障害のある方に月額 20,000円（1口）または 40,000円（2口）を終身支給

保護者が毎月掛金を納め、保護者が亡くなった後（または重度障害になった後）に、障害のある子に毎月年金が支給される制度です。

福祉サービス

生活福祉資金貸付制度（障害者世帯）

用途により異なる。福祉費: 最大580万円、教育支援費: 月額6.5万円以内（大学）、緊急小口資金: 10万円以内。連帯保証人がいれば無利子、いなくても年1.5%

障害のある方がいる世帯に対して、医療費・教育費・住宅改修費等を低金利（または無利子）で貸し付ける制度です。
