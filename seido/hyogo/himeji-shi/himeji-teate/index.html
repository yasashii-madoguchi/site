姫路市障害者福祉金

姫路市内1年以上居住の身体1〜4級・療育A・B1・精神1〜3級の方に4区分で年額10,000〜30,000円が支給されます。所得制限は本人250万円+33万円×扶養親族数。年2回9月25日・3月25日支給。

金額・負担額

1級(身1級・療育A・B1・精1級): 年30,000円 / 2級(身2級・精2級): 年23,000円 / 3級(身3級・精3級): 年15,000円 / 4級(身4級): 年10,000円（年2回 9月25日・3月25日に半期分ずつ支給）

対象
姫路市内に1年以上居住する方で、身体障害者手帳1〜4級／療育手帳A・B1／精神障害者保健福祉手帳1〜3級のいずれかの所持者。本人の所得が基準(250万円+33万円×扶養親族数)以下。
申請先
姫路市健康福祉局 障害福祉課（市役所本庁舎1階・079-221-2111）または家島事務所/夢前・安富・香寺保健福祉サービスセンター
必要書類
障害者福祉金受給資格認定申請書（市公式サイトでPDF配布）、身体障害者手帳・療育手帳・精神障害者保健福祉手帳のいずれか
注意事項
市内1年以上居住要件あり。所得制限: 本人250万円+33万円×扶養親族数(16〜23歳特定扶養+12万円、70歳以上同一生計配偶者・老人扶養+5万円)。給与・公的年金所得の合計から10万円控除して判定。
公式サイト
https://www.city.himeji.lg.jp/kurashi/0000010024.html

姫路市は神戸市・明石市と異なり「対象範囲を広く・金額を小さく」した設計です。神戸市・明石市の介護手当は重度のみ対象で「サービス未利用」要件があり実用ハードルが高い一方、姫路市は軽度〜中度でも対象でサービス利用要件もないため、実用的に受給しやすい設計となっています。

この制度をくわしく

この制度とは

姫路市が市の条例で実施している市単独の現金給付制度です。等級カバー範囲が近隣で最も広い（身4級・療育B1・精神3級まで対象）点が特徴で、軽度〜中度の手帳所持者でも対象になり得る設計です。

一方、年額（最大3万円）は神戸市の重度心身障害者介護手当（年12万円）・明石市（年10万円）と比べると小さく、対象範囲の広さと金額の手厚さがトレードオフの設計となっています。

対象となる方

姫路市内に1年以上居住する方で、以下のいずれかの手帳所持者が対象です。

- 身体障害者手帳 1級・2級・3級・4級

- 療育手帳 A・B1

- 精神障害者保健福祉手帳 1級・2級・3級

1年以上居住要件があるため、姫路市に転入したばかりの方は対象外となります。

所得制限

本人の所得が以下の基準額を超える場合は不支給となります。

基準額の算出

基準額 = 250万円 + 33万円 × 控除対象配偶者・扶養親族数

加算

- 16〜23歳の特定扶養親族 1人につき +12万円

- 70歳以上の同一生計配偶者・老人扶養親族 1人につき +5万円

計算方法

給与所得・公的年金所得の合計から10万円控除して判定。

例: 扶養親族2人（うち高校生1人）の場合

基準額 = 250万円 + 33万円×2 + 12万円×1 = 328万円

他の手当との併用

- 国の障害児福祉手当（月16,560円）: 公式に明記なし、要窓口確認

- 国の特別児童扶養手当: 公式に明記なし、要窓口確認

- 国の特別障害者手当: 公式に明記なし、要窓口確認

- 兵庫県重度障害者医療費助成: 別制度として併用可と推察

- 国の児童手当: 一般的に併給可能

施設入所中の取扱い

施設入所中の取扱いは2026-04-24時点で公式に明記がありません。在宅要件の有無も明示されていないため、入所予定の方は窓口で要確認です。

申請方法と支給時期

姫路市役所本庁舎1階の障害福祉課または出先窓口（家島事務所、夢前・安富・香寺保健福祉サービスセンター）で受付しています。

申請窓口

- 姫路市健康福祉局 障害福祉課: 市役所本庁舎1階

- 電話: 079-221-2111（市代表）

- 出先窓口: 家島事務所、夢前・安富・香寺保健福祉サービスセンター

認定後、年2回（9月25日・3月25日）に半期分ずつ振り込まれます。

20歳到達時の取扱い

本福祉金は年齢上限の明示がなく、20歳到達後も対象継続と推察されます（公式未記載）。20歳到達後も本人の所得が基準額以下であれば継続して支給されます。

申請の手順

  1. 1姫路市内に1年以上居住していることを確認（転入後1年未満は対象外）
  2. 2お子さんが対象等級（身1〜4級／療育A・B1／精神1〜3級）のいずれかに該当することを確認
  3. 3本人の所得が基準額（250万円+33万円×扶養親族数）以下であることを確認
  4. 4姫路市役所本庁舎1階の障害福祉課または出先窓口（家島事務所、夢前・安富・香寺保健福祉サービスセンター）に来所し、障害者福祉金受給資格認定申請書（公式PDF）と障害者手帳を提出
  5. 5市の認定後、年2回（9月25日・3月25日）に半期分ずつ振り込まれる

知っておくと役立つこと

  • ●重要: 姫路市は身4級・療育B1・精神3級まで対象という近隣で最も広い等級カバー範囲です。軽度〜中度の手帳所持者でも対象になり得ます
  • ●ただし金額は最大年30,000円と神戸市（年120,000円）・明石市（年100,000円）の介護手当と比べて小さく、対象範囲と金額のトレードオフ設計です
  • ●1年以上居住要件があります。姫路市に転入したばかりの方は1年経過後に申請可能となります
  • ●所得制限の基準額は本人の所得で判定（250万円+33万円×扶養親族数）。お子さんは通常無収入で要件クリア
  • ●支給は年2回（9月25日・3月25日）に半期分ずつ振り込まれます。年間スケジュールを意識して申請してください
  • ●国の障害児福祉手当（月16,560円・年198,720円）の方が本福祉金より大幅に高額です。重度のお子さんはまず国手当を優先してご検討ください

この制度と一緒に検討されることが多い制度

  • 手当国の制度

    障害児福祉手当

    重度の障害があり日常生活で常時介護が必要な在宅の20歳未満の児童に支給されます。特別児童扶養手当との併給が可能です。

  • 手当国の制度

    特別児童扶養手当

    身体または精神に中度以上の障害がある20歳未満の児童を養育する方に支給されます。

  • 医療費助成（都道府県独自）兵庫県独自

    重度障害者医療費助成事業

    身体手帳1-2級、療育手帳A、精神手帳1級（一般医療のみ）の方の医療費自己負担が軽減されます。自己負担額が明確に規定。
