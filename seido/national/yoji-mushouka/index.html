療育サービス国の制度

幼児教育・保育の無償化

3〜5歳児の幼稚園・保育所・認定こども園等の利用料が無償になります。0〜2歳児は住民税非課税世帯が対象。

金額・負担額

3〜5歳: 利用料無償 / 0〜2歳（住民税非課税世帯）: 利用料無償 / 認可外保育施設: 3〜5歳 月額3.7万円まで、0〜2歳非課税 月額4.2万円まで

対象
3〜5歳児は全世帯。0〜2歳児は住民税非課税世帯。幼稚園・保育所・認定こども園・地域型保育・企業主導型保育・認可外保育施設等。
申請先
施設経由で手続き（認可保育所・幼稚園等）、または区市町村の子育て支援担当窓口
必要書類
施設による。認可外は別途「施設等利用給付認定」が必要
注意事項
障害児通所支援（児童発達支援・放課後等デイ等）も3〜5歳は無償化対象で併用可能。給食費・通園送迎費は対象外。

この制度をくわしく

この制度とは

3〜5歳の全世帯と0〜2歳の住民税非課税世帯を対象に、幼稚園・保育所・認定こども園等の利用料が無償化される国の制度です。2019年10月に開始されました。

障害のあるお子さんにとって、児童発達支援・保育所等訪問支援等の障害児通所支援も3〜5歳は無償化対象となる点が重要です。

対象施設と無償化の範囲

施設の種類により無償化の範囲が異なります。

施設3〜5歳0〜2歳（非課税世帯）
幼稚園（新制度）無償—
幼稚園（新制度未移行）月額25,700円まで—
保育所・認定こども園無償無償
地域型保育（小規模保育等）無償無償
企業主導型保育無償（標準利用料）無償
認可外保育施設等月額37,000円まで月額42,000円まで
児童発達支援等（障害児通所支援）無償—
幼稚園の預かり保育月額11,300円まで（認可外と合算）—

障害のあるお子さんの場合の重要ポイント

無償化と障害児通所支援を併用できる点が、障害のあるお子さんの家庭にとって重要です。

3〜5歳児は「児童発達支援」も無償化対象
  • 児童発達支援事業所・センター
  • 医療型児童発達支援
  • 居宅訪問型児童発達支援
  • 保育所等訪問支援

上記は3〜5歳児は利用料0円で利用できます。

保育所等との並行通園・並行利用
  • 保育所や幼稚園に通いながら、児童発達支援事業所にも通う「並行通園」が可能
  • 両方とも無償（3〜5歳）
  • 加配保育（保育所での補助保育士配置）も自治体独自制度で対応
医療型児童発達支援・医療的ケアの必要な子どもの場合
  • 医療的ケアの必要な子どもも無償化対象
  • 医療費は別途自立支援医療・子ども医療費助成等で軽減

無償化の「対象外」費用（要注意）

無償化されるのは利用料（基本保育料）のみです。以下は対象外のため、引き続き自己負担が必要です。

対象外の費用
費用目安
食材費（給食費）月4,500〜7,500円
通園送迎費月2,000〜5,000円
行事費（運動会・遠足等）年数千〜数万円
制服代・学用品費年1〜3万円
延長保育料（一部施設）月数千円〜
PTA会費・父母会費月数百〜千円

食材費は低所得世帯（年収360万円未満相当）は副食費が免除されます。

0〜2歳児の非課税世帯要件

0〜2歳児の無償化は住民税非課税世帯が対象です（全世帯ではありません）。

住民税非課税の目安
扶養家族年収の目安
0人約100万円以下
1人約156万円以下
2人約205万円以下
3人約255万円以下

課税世帯の0〜2歳児は通常の保育料（所得に応じて月数千〜数万円）がかかります。

認可外保育施設の無償化（要注意）

認可外保育施設を利用する場合、無償化を受けるには別途手続きが必要です。

手続きの流れ
  1. 施設等利用給付認定を市区町村に申請（事前認定が必要）
  2. 認可外保育施設で利用（全額払い）
  3. 領収書を市区町村に提出
  4. 償還払いで還付（月額3.7万円または4.2万円まで）

対象となる認可外保育施設は都道府県に届出をした施設に限ります。認可・届出のない施設は対象外です。

申請方法

認可保育所・認定こども園・新制度幼稚園では自動的に無償化が適用されます。別途申請は不要（施設を通じて市区町村が処理）。

認可外・新制度未移行幼稚園の預かり保育は、施設等利用給付認定が必要です。

施設等利用給付認定の手続き
  1. 市区町村の子育て支援担当窓口で申請
  2. 住民税非課税証明書（0〜2歳児の場合）を提出
  3. 就労証明書等を提出
  4. 認定証を受け取る

副食費（おかず代）の減免

給食の副食費は原則自己負担ですが、以下の世帯は副食費免除されます。

  • 年収360万円未満相当の世帯
  • 第3子以降（年収に関係なく）

主食費（ご飯・パン）は引き続き自己負担です。

他制度との併用

  • 児童発達支援: 3〜5歳は無償化対象、通所受給者証で併用
  • 保育所等訪問支援: 3〜5歳は無償化対象
  • 児童手当: 併給可（独立した制度）
  • 自治体独自の保育料軽減: 併用可能（自治体による）
  • 認可保育所の兄弟姉妹割引: 自治体独自制度と併用可

こんな場合はどうなる

  • 認可外と認可の併用: 認可保育所が優先、認可外は対象外
  • 病児保育・一時預かり: 一定の条件下で対象（市区町村に確認）
  • ベビーシッター: 一定の条件下で対象
  • 3歳になる年度: 4月1日時点で3歳に達していれば無償化対象
  • 幼稚園の預かり保育: 11,300円/月まで（認可外と合算）
  • 引越し: 転出先の市区町村で新たに認定手続き

申請の手順

  1. 1認可保育所・認定こども園・新制度幼稚園: 自動的に無償化適用（別途申請不要）
  2. 2認可外保育施設・新制度未移行幼稚園の預かり保育: 市区町村で「施設等利用給付認定」を申請
  3. 3必要書類: 認定申請書、住民票、就労証明書、住民税非課税証明書（0〜2歳児の場合）
  4. 4認定証を受け取る
  5. 5施設で全額払い後、領収書を市区町村に提出して償還払い（認可外）

知っておくと役立つこと

  • ●障害のある3〜5歳のお子さんは、児童発達支援も無償化対象です。保育所＋児童発達支援の並行通園が可能です
  • ●無償化されるのは利用料のみ。給食費・通園送迎費・行事費等は引き続き自己負担です
  • ●低所得世帯（年収360万円未満相当）・第3子以降は給食の副食費が免除されます
  • ●認可外保育施設を利用する場合は、事前に「施設等利用給付認定」を受ける必要があります
  • ●0〜2歳児の無償化は住民税非課税世帯のみ対象です。課税世帯は通常の保育料がかかります
  • ●医療的ケアの必要な子どもも無償化対象です。医療型児童発達支援・居宅訪問型児童発達支援も利用可能

この制度と一緒に検討されることが多い制度

  • 療育サービス国の制度

    児童発達支援

    未就学児（0〜6歳）向け。日常生活の基本動作や集団適応の訓練を受けられます。3〜5歳は無償化対象。

  • 療育サービス国の制度

    保育所等訪問支援

    保育所・幼稚園・学校等に専門職が訪問し、集団生活への適応のための専門的支援を行うサービスです。

  • 療育サービス国の制度

    居宅訪問型児童発達支援

    重度障害等により外出が著しく困難なお子さんの自宅に専門職が訪問して発達支援を行うサービスです。
