柏市福祉手当

身体・知的・精神の手帳所持者向けに6区分で月額11,000〜14,500円が支給されます。児童（重度）月額14,500円は千葉県内でも最高水準。年3回（4月・8月・12月）支給。

金額・負担額

児童（重度）月14,500円 / 児童（中度）月13,000円 / 寝たきり身体障害者・在宅重度知的障害者 各月12,500円 / その他成人・精神障害者 各月11,000円（年3回 8月・12月・4月に4か月分まとめて支給）

対象
柏市居住。児童重度=身1・2級または重度知的（20歳未満）、児童中度=身3・4級または中度知的（20歳未満）、寝たきり身体=身1・2級でおおむね6か月以上臥床（65歳未満・介護保険未受給）、在宅重度知的=重度判定（介護保険未受給）、精神=精神1級。
申請先
柏市役所障害福祉課（本庁舎別館2階・04-7167-1136）
必要書類
申請書、障害者手帳、預金通帳
注意事項
特別養護老人ホーム・有料老人ホーム等の施設入所、市外居住は対象外。グループホーム・介護老人保健施設・短期入所・介護医療院（市内）は対象継続。

柏市は千葉県内中核市の中で月額が最高水準で、児童（重度）14,500円は政令市の千葉市（月7,000円）の約2倍にもなります。所得制限の明記がないため、対象等級に該当する場合は必ず申請を検討してください。船橋市・千葉市は「国手当受給者は対象外」の明文があるため、重度のお子さんは国手当（月16,560円）と本手当（月14,500円）の併給可否を窓口で確認することが重要です。

この制度をくわしく

この制度とは

柏市が市の条例で実施している市単独の現金給付手当です。児童（重度）月額14,500円は千葉県内の市単独障害のある子ども手当の中で最高水準で、隣接する船橋市（月8,000円）・市川市（単独8,000円）・流山市（月8,650円）と比較しても明らかに手厚い設計です。

身体・知的・精神・寝たきりを6区分に細分化し、年齢（児童/成人）と障害程度・状態で金額を変えるきめ細かい設計が特徴です。

対象となる方

柏市内に居住している方で、上記の6区分のいずれかに該当する方が対象です。

- 児童（重度）: 20歳未満で身体障害者手帳1・2級、または重度判定の知的障害児

- 児童（中度）: 20歳未満で身体障害者手帳3・4級、または中度判定の知的障害児

- 寝たきり身体障害者: 身体1・2級で居宅において寝たきりの状態がおおむね6か月以上、65歳未満、介護保険未受給

- 在宅重度知的障害者: 重度以上と判定された在宅の知的障害者、介護保険未受給

- その他成人: 身体1・2級または中度以上の知的障害者

- 精神障害者: 精神障害者保健福祉手帳1級

所得制限

所得制限の有無は2026-04-24時点で公式ページに明記がありません。同種の市単独手当としては所得制限を設けない例もありますが、申請前に柏市障害福祉課（04-7167-1136）にてご確認ください。

他の手当との併用

- 国の障害児福祉手当（月16,560円）: 公式に明記なし、要窓口確認

- 国の特別児童扶養手当（1級58,450円／2級38,930円）: 公式に明記なし、要窓口確認

- 国の特別障害者手当（月29,590円）: 公式に明記なし、要窓口確認

- 千葉県重度心身障害者医療費助成: 公式に併給言及なし、別制度として併用可と解される

- 国の児童手当: 一般的に併給可能

施設入所中の取扱い

特別養護老人ホーム・有料老人ホーム等への施設入所、または居住の実態が柏市外にある場合は対象外となります。

ただし以下は対象継続となります（重要）。

- グループホーム（市内）

- 介護老人保健施設

- 短期入所（ショートステイ）

- 介護医療院（市内）

20歳到達時の取扱い

児童区分（重度・中度）から成人区分（その他成人・寝たきり等）への切替が想定されますが、自動切替か再申請かは2026-04-24時点で公式に明記がありません。20歳到達前に区分変更手続きの要否を窓口でご確認ください。

申請の手順

  1. 1お子さんが児童（重度）または児童（中度）の対象等級に該当することを確認（20歳未満）
  2. 2施設入所（特養・有料老人ホーム等）でないこと、市外居住でないことを確認
  3. 3柏市役所障害福祉課（本庁舎別館2階）に来所し、申請書・障害者手帳・預金通帳を持参
  4. 4市の認定後、年3回（8月・12月・4月）にまとめて4か月分（重度58,000円・中度52,000円）が振り込まれる
  5. 520歳到達時に成人区分への切替手続きが必要になる可能性あり（要窓口確認）

知っておくと役立つこと

  • ●重要: 児童（重度）月額14,500円は千葉県内で最高水準です（船橋・市川・流山・千葉各市は8,000〜10,500円）。柏市にお住まいの重度のお子さんは見落とさず申請してください
  • ●支給は年3回（8月・12月・4月）に4か月分まとめて振り込まれます。最低でも年4か月分（重度58,000円・中度52,000円）が安定収入として計画立てやすい設計です
  • ●グループホーム（市内）・短期入所・介護老人保健施設・介護医療院（市内）の利用は対象継続となります。レスパイト目的の短期入所利用も問題ありません
  • ●寝たきり身体区分は「65歳未満・介護保険未受給」が要件で、介護保険を受けると対象外となります。65歳到達時の取扱いは要窓口確認です
  • ●所得制限の有無や国の障害児福祉手当との併給可否は公式に明記がありません。月額が高水準のため、申請前に必ず障害福祉課（04-7167-1136）に併給可否をご確認ください
  • ●中度区分でも月13,000円と政令市の同種制度（横浜廃止・川崎は別設計）と比べても手厚い水準です。身3-4級または中度知的のお子さんも対象になり得ます

この制度と一緒に検討されることが多い制度

  • 手当国の制度

    障害児福祉手当

    重度の障害があり日常生活で常時介護が必要な在宅の20歳未満の児童に支給されます。特別児童扶養手当との併給が可能です。

  • 手当国の制度

    特別児童扶養手当

    身体または精神に中度以上の障害がある20歳未満の児童を養育する方に支給されます。

  • 医療費助成（都道府県独自）千葉県独自

    重度心身障害者（児）医療給付改善事業

    身体手帳1-2級、療育手帳○A・A1・A2、精神手帳1級（令和2年8月〜）の方の医療費自己負担が1回300円に軽減されます。平成27年8月から現物給付化。
